How did I make this blog?

😁Good evening, everyone.

📝This is my first post on this blog made by Gridea. Everything is free except for this domain name, which I paid for, so this is almost a zero cost blog. If you want to make a blog like me, I highly recommend you use Gridea, it is an easy to use program and there are many tutorials online to help you use it.

Why did I make this blog?

🙋‍♂️As you know, I already have a blog, built using WordPress, but it is geared towards Chinese people and is a Chinese blog. I would like to build an English blog for posting my articles written in English. On the one hand, I can improve my English writing skills, and on the other hand, I hope to meet foreign friends through it and communicate and learn with them.

👉I wanted to build a blog with minimal cost and gridea was my first choice.

📌Gridea is a static blog writing client developed by EryouHao. I used it with GitHub Pages and it worked perfectly to build a static blog with a domain name purchased from namesilo.

What will I write on this blog?

👀Because I have limited energy, I may not post as often.

🎈I will be posting my thoughts and reflections on this blog, based on my level of English writing, and of course, I hope that in the future, my level of English will allow me to write fluently in English.
